1. Pipeline Architecture

The SEI Harmonization Pipeline transforms 7 monthly geospatial source tables (15 indicators) into two robust, harmonized indices through a 4-phase process.

Phase 1

Pre-Processing
  • Quarterly aggregation
  • Targeted imputation
  • STL decomposition

Phase 2

Weight Extraction
  • PCA branch (PC1)
  • ICW branch (inv-cov)
  • Unity normalization

Phase 3

Synthesis
  • Meta-PCA fusion
  • Envelope weighting
  • Min-Max scaling

Phase 4

Output
  • SEI_METAPCA_FINAL
  • SEI_ENVELOPE_FINAL
  • Write to Snowflake

2. Methodology

PCA Branch (Infrastructure-Heavy)

  1. Compute covariance matrix of MinMax-scaled trend lines
  2. Eigen-decompose to extract PC1 factor loadings
  3. Flip sign if loading sum < 0 (directional alignment)
  4. Normalize: W_PCA = loadings / sum(loadings) → sum = 1
  5. Score = weighted sum; Min-Max scale to [0,1]
PCA assigns higher weights to correlated infrastructure indicators (LST, CROPS, BARE dominate with >35% each).

ICW Branch (Environment-Sensitive)

  1. Compute covariance matrix of MinMax-scaled trend lines
  2. Invert the covariance matrix (pseudo-inverse if singular)
  3. Row-sum the inverse covariance matrix
  4. Normalize: W_ICW = |rowsums| / sum(|rowsums|) → sum = 1
  5. Score = weighted sum; Min-Max scale to [0,1]
ICW penalizes redundancy — unique indicators (SCHOOL_CT=26%, BANK_CT=14%, NDVI=13%) receive higher weight.

Branch A: Meta-PCA

  1. Stack SEI_PCA and SEI_ICW into 2D matrix
  2. Standardize (z-score) both columns
  3. Compute 2×2 covariance → extract Meta-PC1
  4. Directional alignment (flip if sum < 0)
  5. Project onto Meta-PC1 → Min-Max → SEI_METAPCA_FINAL

Finds the axis of maximum joint variance between the two sub-indices.

Branch B: Envelope Weighting

  1. W_env(i) = max(W_PCA(i), W_ICW(i)) per indicator
  2. Unity normalization: W_norm = W_env / sum(W_env)
  3. Matrix multiply: weights × scaled trends
  4. Min-Max scale → SEI_ENVELOPE_FINAL

Never discards information — always picks the stronger weight per indicator.

Unity Constraint: All weight vectors (PCA, ICW, Envelope) are explicitly normalized to sum to exactly 1.0.

Drives Envelope Higher

These indicators have high PCA weights (picked by max operation) but low ICW weights:

  • LST_MED (r=-0.962): PCA=0.366, ICW=0.042
  • LST_MEAN (r=-0.960): PCA=0.364, ICW=0.005
  • MEAN_CROPS (r=-0.868): PCA=0.378, ICW=0.028

PCA assigns >35% weight to temperature/crops (correlated cluster). Envelope inherits directly.

Drives MetaPCA Higher

These have high ICW weights or negative PCA weights:

  • NDVI_MEAN (r=+0.778): PCA=-0.249, ICW=0.132
  • RAIN_MED (r=+0.250): PCA=-0.032, ICW=0.077
  • HOSP_CT (r=+0.222): PCA=-0.020, ICW=0.043

NDVI has a negative PCA weight. MetaPCA captures this via meta-synthesis.

Root Cause

  1. Weight concentration: PCA creates a highly concentrated weight vector dominated by LST and CROPS (>35% each). ICW distributes toward unique signals (SCHOOL=26%, BANK=14%).
  2. Envelope inherits PCA's concentration: max(W_PCA, W_ICW) inherits PCA's large LST/CROPS weights, making Envelope temperature/agriculture-driven.
  3. MetaPCA partially cancels concentration: By operating on two sub-index scores, MetaPCA finds a compromise that neither method dominates.
  4. NDVI sign conflict: PCA treats vegetation as inversely related to development (W=-0.25), while ICW gives it a positive weight (W=+0.13). This is the single largest driver of divergence.

Recommendations

Use SEI_ENVELOPE for policy where all dimensions matter equally.
Use SEI_METAPCA for research where statistical consensus is preferred.
Report both to demonstrate robustness (or lack thereof) in rankings.

Limitations

  1. Relative scale: Cannot compare across states or time windows without re-fitting
  2. Census codes: District identifiers are PC11_SD_ID codes (not human-readable names)
  3. Temporal coverage: LULC limited to 2015–2024; edge effects possible
  4. STL assumptions: Period=4 assumes annual seasonality; sub-annual structural breaks may be smoothed
  5. Negative correlation: The -0.557 correlation indicates methods capture fundamentally different signals
